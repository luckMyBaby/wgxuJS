e.g: 提示框
代码： 
					<script>
						$('#btn1').on('click', function(){
					    	NUI.NUIArtTips({
					    		width: 250,
					    		height: 300,
					    		content: '<div style="300px;">我是html</div>',
					    		ok: function(){
					    			alert('只有确定按钮');
					    		},
					    		cancel: function(){
					    			alert('你点击了取消');
					    		}
					    	});
					    });
					</script>

NUIArtTips、NUIArtSubmit、HBArtIframe

属性 参数 说明
需在NUI.min.js 之前引入 plugins/nui/plugins/dialog/dialog.js
width 默认 auto, 可设置固定或百分比 弹出框内容宽度
height 同上 弹出框内容高度
content 一段 html 字符串 详细数据可参见demo示例中数据
ok 回调函数, 点击确定之后触发 详细数据可参见demo示例中数据)
okVal 确定按钮名称默认值 确定 详细数据可参见demo示例中数据)
cancel 回调函数, 点击取消之后触发 详细数据可参见demo示例中数据)
cancelVal 取消按钮名称默认值 取消 详细数据可参见demo示例中数据)

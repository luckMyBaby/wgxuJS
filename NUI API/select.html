NUISelect插件

示例:
代码： 
                    <select id="select" name="citylist"  class="pos NUISelect" data-options="{width:200}">
                        <option value="">请选择</option>
                        <option value="Test21">安徽省</option>
                        <option value="Test15">江苏省</option>
                        <option value="Test41">浙江省</option>
                        <option value="Test31">福建省</option>
                        <option value="Test51">广东省</option>
                        <option value="Test31">福建省</option>
                        <option value="Test51">黑龙江省</option>
                    </select>

NUITab API列表

属性 参数 说明
NUISelect - 申明NUISelect类就可以使用框架提供的组件
data-options

1、width(组件宽度)

2、valueColor(字体颜色，默认黑色)

3、beforeSelect函数，选中前函数

4、onSelect函数，选中时的函数

5、afterSelect选中后的函数

 组件对外提供的接口：

1、width(组件宽度)

2、valueColor(字体颜色，默认黑色)

3、beforeSelect函数，选中前触发的函数

4、onSelect函数，选中时触发的函数

5、afterSelect选中后触发的函数

备注：1、该插件会自动把原生的select组件替换成新的UI组件，在新的UI组件上操作会同步到原生select上，所以后台获取值的时候，可以直接获取原生的值。如$(selectName).val();,所以后台人员可以直接操作原生的select。
2、可以直接使用class来控制原生的select布局，这个布局会同步到UI框架中。

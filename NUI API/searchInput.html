NUISearchInput插件

示例:
代码： 
                    <input type="text" class="NUISearchInput NUISearchAction" data-url="./data/Linkage.json" data-params = "value" data-options="{IDField:'ID',textField: 'text'}">
回显示例:
代码： 
                    <input type="text" class="NUISearchInput NUISearchAction" data-url="./data/Linkage.json" data-params = "value" data-options="{IDField:'ID',textField: 'text'}" data-defaultVal="北京市">
说明：

该插件适用场景，匹配用户输入的内容，如果有则给与提示，如果没有则提示未匹配。类似于百度搜索框，因实时监控用户输入内容，所以对性能要求较多。请根据实际使用场景使用。

NUITab API列表

属性 参数 说明
NUISearchInput - 申明该class表明仅使用搜索输入框的样式
NUISearchAction - 申明该class表明仅使用搜索框输入的交互
data-url 是一个字符串,这个url即匹配用户输入内容的接口 详细数据可参见demo示例中数据
data-parmas 是一个字符串，是搜索框传到后台的参数 详细数据可参见demo示例中数据
data-options

1、IDField(数据的key，如id，uuid等）

2、textField(数据的id对应的值，如北京、上海等）

 详细数据可参见demo示例中数据)
data-defaultVal 一个字符串，回显用户之前输入的字符串。 详细数据可参见demo示例中数据)

NUITab API列表

属性 参数 说明
NUITabArea - 该类是NUITab插件的dom容器
NUITab - 申明NUITab类，即使用框架的样式
NUITabAction - 申明NUITabAction类，即使用NUI框架的交互

备注：如使用NUITab插件，请务必保证使用demo的dom结构，否则插件不生效。NUITab组件采用样式和交互分离的模式，即看最大化的根据用户需求来使用该框架，申明NUITab类，表明采用框架样式，申明NUITabAction类，表明使用框架的交互。如果两者都申明，则表明可以使用框架的样式和交互。

NUIBtns插件

e.g:
代码： 
                    <button class="BtnNormal">默认</button>
                    <button class="BtnNormal BtnCancel">取消</button>
                    <button class="BtnNormal BtnOk">确定</button>
                    <button class="BtnNormal BtnDisabled">禁用</button>
                    <button class="BtnMin BtnNormal">默认(缩小)</button>
                    <button class="BtnMin BtnNormal BtnCancel">取消(缩小)</button>
                    <button class="BtnMin BtnNormal BtnOk">确定(缩小)</button>
                    <button class="BtnMin BtnNormal BtnDisabled">禁用(缩小)</button>

NUI按钮API列表

属性 参数 说明
BtnCancel - NUI取消按钮样式
BtnOk - NUI确定按钮样式
BtnDisabled - NUI禁止按钮样式
BtnMin - NUI缩小按钮样式

备注：按钮功能比较单一，框架没有进行交互设计，如demo所示，可以同时赋多个类来进行申明样式。如需对框架样式进行修改的话，请在类后面，添加自定义的class来进行样式控制。

e.g： 
<button class="BtnNormal BtnDisabled className">禁用</button>

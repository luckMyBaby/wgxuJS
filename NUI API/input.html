NUIInput插件

e.g:
代码： 
<input type="text" class="NUIInput">

描述：输入框比较单一，框架没有设计交互，如需对框架样式进行修改的话，请在NUIInput类后面，添加自定义的class来进行样式控制。

e.g： 
<input type="text" class="NUIInput className">

NUITableAjax

属性 参数 说明
NUITableAjax - 申明该class表明使用框架表格数据渲染插件
data-url 是一个字符串,这个url即及请求后台表格数据的接口 详细数据可参见demo示例中数据
data-filed 是一个字符串，请求数据需要输出的属性 详细数据可参见demo示例中数据
data-format 是一个函数名的字符串, 后台自定义函数名; 该函数有3个参数

arguments[0]: 当前行, 整行数据

arguments[1]: 当前行tr jquery对象

arguments[2]: 当前行该函数的td jquery对象

 详细数据可参见demo示例中数据)

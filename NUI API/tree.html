    说明：

    NUI.NUITree继承$.fn.zTree 所以调用方法和zTree一样 例
    $.fn.zTree.init("#sTree", {}, zNodes);
    NUI.NUITree.init($("#sTree"), {}, zNodes);

    使用树参见时需要已下几个要求：

    1、框架集成zTree插件，故在使用前，请先引入zTree的css和js文件。
    路径：
    　　　plugins/nui/plugins/tree/zTreeStyle/zTreeStyle.css
    　　　plugins/nui/plugins/tree/ztree.min.js (需再NUI.min.js之前引入)

    2、在树节点上需添加class="zTree"添加样式

    3、如果了解更多zTree规则，请点击这里
